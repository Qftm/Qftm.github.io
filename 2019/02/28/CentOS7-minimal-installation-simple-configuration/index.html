系统配置

配置动态获取IP

1、修改配置文件

vi /etc/sysconfig/network-scripts/ifcfg-eno33 //eno33是你的网卡名字

2、修改选项ONBOOT //默认ONBOOT处于关闭状态

ONBOOT = no ==>> ONBOOT = yes

3、重启服务

systemctl restart network

4、查看IP信息

ip addr

配置静态IP

1、修改配置文件

vi /etc/sysconfig/network-scripts/ifcfg-eno33 //eno33是你的网卡名字

2、添加配置

ONBOOT=yes             #设置启动
BOOTPROTO=static       #启用静态IP地址
IPADDR=192.168.2.11    #设置静态IP
NETMASK=255.255.255.0  #设置掩码
GATEWAY=192.168.2.1    #设置网关IP
DNS1=8.8.8.8           #设置DNS

安装ifconfig命令

1、默认没有安装ifconfig命令

2、安装

查看包含ifconfig命令的组件
yum provides ifconfig 
或 
yum whatprovides ifconfig 

安装
yum install -y net-tools

3、检测ifconfig命令

vim的安装

vim是vi的增强版

1、默认只有vi编辑器

2、安装

查看vim安装组件
yum provides vim

安装
yum install -y vim-enhanced

wget的安装

Wget主要用于下载文件，在安装软件时会经常用到

1、查看wget安装组件 

yum provides wget

2、安装wget

yum install -y wget
